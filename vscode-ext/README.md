# Aura-NX VS Code Extension

Aura-NX Development Tools for VS Code.

## Features

- **Targets View**: Discovers and lists Aura-NX compatible devices on the network.
- **Screen Capture**: Quickly capture the screen of the active target.

## Commands

- `Aura: Capture Screen`: Captures the current framebuffer of the foreground game on the selected device.

## Requirements

- [Aura-NX MCP Server](https://github.com/aura-nx/aura-nx) must be running and configured.

## Extension Settings

This extension contributes the following settings:

* `aura.mcpServerPath`: Path to the Aura-NX MCP server executable/script.
